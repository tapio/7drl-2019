About

For bugs and feedback, you can create a GitHub issue.

This game was created by Tapio Vierros for the 2019 Seven Day Roguelike Challenge, with the aid of rot.js library and lots of code reuse from my previous 7DRL, Fantastic Dungeons.

✕

Instructions

Navigate by clicking/touching the tile where you want to go. You can also use the keyboard (arrows, numpad, Vi-keys). Bump into stuff to interact. Right click/long press prints info on things in the world.

During play, you can access settings - such as fullscreen and sound toggles - from the upper right corner of the screen.
